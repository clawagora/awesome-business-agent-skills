# Contributing to Awesome Business Agent Skills

Thank you for helping make this list better! We welcome contributions from business owners, developers, and anyone building useful AI agent skills.

## How to Add a Skill

1. **Fork** this repository.
2. **Edit** `README.md` and add your skill to the appropriate category.
3. **Submit** a pull request.

### Skill Entry Format

Each skill entry should follow this format:

```markdown
- **Skill Name** - One or two sentences describing what it does and why it's useful for a business. `openclaw` `hermes`
```

- Start with a bold skill name.
- Follow with a clear, concise description (1-2 sentences).
- End with runtime tags: `openclaw`, `hermes`, or both.
- If the skill links to a public repo or marketplace page, link the skill name.

### Example

```markdown
- **[Invoice Processor](https://example.com/invoice-skill)** - Extract line items, amounts, and dates from incoming invoices (PDF, email). Match against purchase orders and flag discrepancies. `openclaw`
```

## Quality Standards

We want this list to be genuinely useful, not a dumping ground. Before submitting, make sure your skill:

- **Solves a real business problem** - Would a non-technical business owner care about this?
- **Is production-ready** - The skill should actually work, not be a proof-of-concept.
- **Has a clear description** - Someone should understand what it does in 10 seconds.
- **Isn't a duplicate** - Check that a similar skill isn't already listed.
- **Is in the right category** - Place it where a business owner would look for it.

## Adding a New Category

If your skill doesn't fit any existing category, you can propose a new one. Open an [issue](https://github.com/clawagora/awesome-business-agent-skills/issues) first to discuss whether a new category makes sense.

## Reporting Issues

- **Broken link?** Open an issue using the "Report Broken Link" template.
- **Outdated skill?** Open an issue or PR to update the entry.
- **Spam or low quality?** Open an issue and we'll review.

## Code of Conduct

This project follows our [Code of Conduct](CODE_OF_CONDUCT.md). By participating, you agree to uphold a respectful and inclusive environment.
